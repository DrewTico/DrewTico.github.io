How to Create an "About Me" Page on GitHub Pages

Step 1

Create a GitHub Account

Go to GitHub Sign Up and sign up. Follow the prompts to create your account by entering your username, email, and password.

Step 2

Create a New Repository

Once logged in, click the "+" icon in the top-right corner, select "New repository", and name it username.github.io (replace "username" with your actual GitHub username).
