How to Create an "About Me" Page on GitHub Pages

Table of Contents

Step 1: Create a GitHub Account & Repository

Purpose

This step will help you set up a GitHub account and create a repository where your website files will be stored.

Welcome! Let’s start by creating a GitHub account (if you don’t have one) and setting up your first repository.

Creating an Account: If you don’t already have a GitHub account, go to GitHub.com and sign up. It's free! Once registered, log in to your new account.

Creating Your Repository: Click on the “New Repository” button on your dashboard after logging in.

Important: Name your repository like this: yourusername.github.io. Replace "yourusername" with your actual GitHub username.
Tip: Make sure the repository is set to public, so your website will be visible to everyone!

Step 2: Set Up GitHub Pages

Purpose

This step will help you configure your GitHub Pages settings to publish your site.

Go to your repository’s settings by clicking on the “Settings” tab. Scroll down until you find the "GitHub Pages" section.

Under “Source”, select the main branch (this is where all your code and changes will go).
It might take up to 5 minutes for your new site to become available. Refresh the page after a few minutes to check.

Step 3: Customize Your 'About Me' Page

Purpose

This step allows you to personalize your page content by editing your default index.html file.

Click on the index.html file in your repository. Replace the placeholder text with your own information.

Pro Tip: Replace the "username" placeholders with your actual GitHub username.

Here’s an example code that you can use to create a simple 'About Me' page. Just copy and paste the following HTML code into your index.html file:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>About Me - [Your Name]</title>
</head>
<body>
    <header>
        <h1>About Me</h1>
    </header>

    <section>
        <h2>Hello! I'm [Your Name]</h2>
        <p>
            Welcome to my personal page! I am a [your profession or hobby] who loves 
            [something you're passionate about]. This website is a small introduction 
            to who I am and what I do.
        </p>
    </section>

    <section>
        <h2>Connect with me</h2>
        <ul>
            <li><a href="https://github.com/yourusername" target="_blank">GitHub</a></li>
            <li><a href="https://linkedin.com/in/yourusername" target="_blank">LinkedIn</a></li>
            <li><a href="https://twitter.com/yourusername" target="_blank">Twitter</a></li>
        </ul>
    </section>

</body>
</html>

Feel free to customize the content by adding more sections or CSS styling later.

Step 4: Commit and Publish

Purpose

Commit your changes to publish your new 'About Me' page live on GitHub Pages.

After editing your index.html file, scroll to the top and click the “Commit changes” button.

Make sure to add a meaningful commit message (e.g., “Updated About Me page”) to keep track of changes.
It may take a minute or two for changes to appear live. Refresh your browser after a bit to see the results.

Troubleshooting

If something isn’t working, try these solutions:

  • Website not showing up: Wait a few minutes, then refresh. It can take time for GitHub Pages to publish the site.
  • Changes not appearing: Double-check that you’ve committed your changes to the main branch.
